# hugo-site-generator

Quickly generate new Hugo site with theme pre-download

* Install NodeJS LTS
* `npm i -g hugo`
* `npm i -g hugo-site-generator`
* `hugo-generate`
    * Answer quesions
    * New site name: will be used as directory name for site generation
    * Theme's Git repository: URL to github theme project 
* `cd <theme_name>`
* `hugo server -D`
